Google FooBar

I completed Google's FooBar challenge April 2023 after a friend of mine sent me a referral link. This involved solving a series of mathematical and/or coding problems using Python 2.7 (or Java). The challenge included 5 levels of increasing difficulty with 1-3 tasks per level. Here I summarise the tasks and explain how I solved each problem.

Challenge 1

Re-ID
=====

Write a function solution(n) which takes in the starting index n of a string of all primes, "2357111317192329...", and returns the next five digits in the string. For example, if n is 3 then their ID number will be "71113". The value of n will always be between 0 and 10000.

Challenge 2

Please Pass the Coded Messages
==============================

You have L, a list containing some digits (0 to 9). Write a function solution(L) which finds the largest number that can be made from some or all of these digits and is divisible by 3. If it is not possible to make such a number, return 0 as the solution. L will contain anywhere from 1 to 9 digits. The same digit may appear multiple times in the list, but each element in the list may only be used once.

-- Test cases --
Input : solution([3, 1, 4, 1])
Output: 4311

Input : solution([3, 1, 4, 1, 5, 9])
Output: 94311
